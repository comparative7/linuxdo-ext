# LinuxDo Auto-Browser

在 [LinuxDo](https://linux.do) 论坛模拟真实用户的自动浏览插件（Chrome Manifest V3）。

**核心原则**：只读不写；运行在已登录的真实浏览器环境中；操作间带随机延迟，降低机器特征。

---

## 功能一览

| 能力 | 说明 |
|------|------|
| 自动浏览 | 在帖子列表识别未读主题并进入阅读 |
| 拟人滚动 | 详情页按随机步长 / 停顿 / 动画滑动 |
| 日限与休息 | 每日帖数上限；连续 N 帖后强制休息 |
| 无未读等待 | 列表暂无未读时随机等待后刷新再扫 |
| 长帖早退 / 部分阅读 | 按概率提前离开，不必每帖滚到底 |
| 今日足迹 | 记录今日已读主题，可清空列表 |
| 重置今日帖数 | 手动将今日帖数清零（不影响新读楼与足迹） |
| 页面 HUD / 工具栏徽章 | 运行状态与今日计数可视化 |

---

## 安装（开发者模式）

本项目**无构建步骤**，直接加载源码目录即可。

1. 用 Chrome 打开 `chrome://extensions/`
2. 打开右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本仓库根目录（含 `manifest.json` 的文件夹）
5. 确认已在浏览器中登录 [linux.do](https://linux.do)

代码更新后，在扩展页点击该扩展的 **重新加载**，再试功能。

---

## 快速开始

1. 打开 LinuxDo 任意**帖子列表页**（如首页、最新），不要停在主题详情页。
2. 点击工具栏扩展图标，打开 Popup。
3. 点击 **开始**。插件会自动点未读帖 → 滚动阅读 → 返回列表 → 循环。
4. 需要中断时点 **停止**。

可选：Popup 点 **设置** 打开配置页，按个人节奏调整日限、休息与滚动参数后保存。

---

## Popup 说明

| 控件 | 作用 |
|------|------|
| 状态行 | 运行 / 停止 / 休息 / 等待，以及「今日 帖数/上限 · 新读 楼」 |
| 今日足迹 | 折叠列表：时间、退出原因、本帖新读楼、标题；点击可打开主题 |
| 清空今日 | 仅清空足迹列表，**不**改变今日帖数与新读楼统计 |
| 开始 / 停止 | 启停自动化；休息或等待中也会禁用「开始」 |
| 跳过休息 / 立即扫描 | 休息中可跳过休息；无未读等待中可立刻刷新扫描 |
| 重置今日帖数 | 将今日 `count` 置 0，解除达限阻塞；**不**改新读楼与足迹 |
| 设置 | 打开独立配置页（Options） |
| 打开 LinuxDo 主页 | 新标签打开论坛首页 |

### 达限与重置

- 今日帖数达到「每日上限」后自动停止，状态会提示已达上限。
- 跨自然日（本地日期）会自动把计数清零。
- 若希望**同一天**继续浏览，可用 **重置今日帖数**（或在设置里临时提高每日上限）。

### 使用注意

- 请在**列表页**点开始；详情页无法作为起点。
- 不要依赖插件做任何发帖、回复、点赞等写操作（本扩展不会也不应实现这些）。
- 滚动与等待均含随机延时，请勿期望“秒刷完站”。

## 功能设置（Options）

入口：Popup → **设置**，或扩展管理页 → 该扩展 → **扩展选项**。

运行中、休息中或无未读等待时，配置页会锁定；需先停止再改。改完点 **保存**；多数参数在下次启动或下一帖后生效。**恢复默认**会还原全部设置项（不清除今日计数与足迹）。

### 日限与休息

| 设置项 | 默认 | 范围 | 说明 |
|--------|------|------|------|
| 每日上限 | 100 帖 | 1–500 | 今日完成数达到后停止；跨天自动重置 |
| 每连续浏览 | 10 帖 | 1–50 | 单次不间断完成这么多帖后进入休息 |
| 休息时长 | 15 分钟 | 1–120 | 休息结束后自动继续下一轮 |

### 滚动（详情页）

均为随机区间：每次在最小值与最大值之间取值。

| 设置项 | 默认 | 说明 |
|--------|------|------|
| 步长 | 50–200 px | 单次滚动像素 |
| 停顿 | 500–2000 ms | 两步之间的等待 |
| 滑动动画 | 400–1400 ms | 单次 smooth scroll 时长 |

步长越大、停顿越短，读帖越快，但也更像机器；建议保持合理区间。

### 阅读策略

| 设置项 | 默认 | 说明 |
|--------|------|------|
| 启用长帖早退 | 开 | 主题过长或阅读过久时，按概率提前离开 |
| 早退楼数阈值 | 80 楼 | 主题总楼数达到该值后可能触发早退判定 |
| 早退时长阈值 | 3 分钟 | 单帖阅读时长达到后可能触发早退判定 |
| 早退概率 | 70% | 满足阈值后真正早退的概率 |
| 启用部分阅读 | 开 | 未必要滚到底，按概率在中段离开 |
| 部分阅读概率 | 35% | 触发“读到某一深度就走”的概率 |
| 部分阅读深度 | 40%–70% | 相对可滚动高度的离开区间 |

早退与部分阅读共用“提前完成”出口；优先级与细节以实现为准。关掉两项则更接近“尽量读完”。

### 页面角标 (HUD)

论坛页右下角只读浮层（不挡点击）。

| 设置项 | 默认 | 说明 |
|--------|------|------|
| 显示页面角标 | 开 | 总开关；关则完全不显示 |
| 显示标题 | 开 | “LinuxDo Bot” |
| 显示运行状态 | 开 | 阅读中 / 扫描中 / 休息与等待倒计时等 |
| 显示今日统计 | 开 | 今日帖数与新读楼 |
| 显示本主题进度 | 开 | 仅详情：已读/总数/本次新增 |

### 工具栏徽章

扩展图标右上角短文字。

| 设置项 | 默认 | 说明 |
|--------|------|------|
| 显示工具栏徽章 | 开 | 总开关；关则清空徽章 |
| 显示今日帖数 | 开 | 运行/停止时的数字；过大显示 `99+` |
| 显示状态字 | 开 | `休` / `等` / `满` 等，通常优先于纯数字 |

### 调试

| 设置项 | 默认 | 说明 |
|--------|------|------|
| 列表扫描调试日志 | 关 | 在开发者工具控制台输出更细的选帖扫描信息 |

## 推荐用法

- **日常保活**：日限 50–150，休息 8–15 帖 / 10–20 分钟，滚动保持默认随机区间。
- **读得更快**：略增大步长、缩短停顿；可适当提高部分阅读概率。仍建议保留休息与日限。
- **排查误点**：看 Popup「今日足迹」；确认后再 **清空今日**（足迹）或 **重置今日帖数**（仅计数）。

---

## 不会做什么（非目标）

- 不自动发帖、回复、点赞、关注、改草稿
- 不处理登录 / 密码（需你事先在浏览器登录）
- 不引入 React / Vue / Webpack 等前端框架或打包工具
